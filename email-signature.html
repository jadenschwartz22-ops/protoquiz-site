ProtoQuiz Email Signature Setup

  1. Click "Copy Signature" below
  2. Go to Outlook Web Signature Settings
  3. Click "+ New signature", name it "ProtoQuiz"
  4. Paste into the signature editor (Cmd+V)
  5. Set as default for new messages AND replies
  6. Save
